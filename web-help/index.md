## Was ist HAM-Tools?

Eine native **macOS-App** für Funkamateure mit Fokus auf:
- **Logbuch** mit allen üblichen Award-Workflows (DXCC, WAZ, WAS, POTA, SOTA, WWFF, BOTA)
- **Contest-Logger** mit Cabrillo V3-Export für die wichtigsten Major-Contests
- **CAT-Anbindung** für Yaesu / Icom / Kenwood / Elecraft via Hamlib
- **Antennen-Rechner** für Drahtantennen, Richtstrahler, Spezialantennen, Spulen

Distribution per **DMG mit Apple-Notarisierung** — Gatekeeper-freundlich. Lizenzsystem
mit eingebauter **Update-Verteilung** (Cmd+Opt+U), Modell *Lifetime + 12 Monate Updates*.

## Schnellstart

1. **DMG herunterladen** und in `/Applications` ziehen
2. **App starten** (beim ersten Öffnen ggf. Systemeinstellungen → Datenschutz)
3. **Lizenz-String** in **Einstellungen → Lizenz** einfügen (oder 50 QSO Demo testen)
4. **Eigenes Rufzeichen** in **Einstellungen → Station** eintragen
5. **Erstes QSO** im Logbuch-Tab loggen

Detaillierter: [Erste Schritte →](/getting-started)
