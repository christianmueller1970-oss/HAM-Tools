# Bug melden

Der schnellste Weg: in der App **Cmd+Shift+B** drücken (oder Menüleiste → **Hilfe → Bug melden…**).

## Was passiert?

Ein Sheet öffnet sich mit Eingabefeldern:

- **Modul-Dropdown** — wähle das Modul (Logbuch, Contest, POTA, DX-Cluster, CAT, …)
- **Kurz-Titel** — was ist passiert (One-Liner)
- **Beschreibung** — was hast du getan, was sollte passieren, was passiert tatsächlich
- **Diagnose anhängen** — Default an, fügt App-Version, macOS-Version, Hardware-Modell und dein Callsign der Mail bei

Klick auf **"Mail öffnen"** → macOS-Mail öffnet sich mit allem vorausgefüllt:

```
An: bugs@funkwelt.net
Betreff: [BUG-20260513-1924-XX] [Modul: Contest] Kurz-Titel
```

Nur **"Senden"** klicken — fertig.

## Was wir mit deinem Report tun

Jeder Report kriegt eine eindeutige ID im Subject (`BUG-yyyyMMdd-HHmm-XX`). Wir antworten dir mit Bestätigung oder Rückfragen unter derselben ID. Behobene Bugs erscheinen im [Changelog](/changelog) und [Bekannte Bugs](/known-bugs).

## Ohne die App

Falls die App selbst gerade unbenutzbar ist:

- **Mail an `bugs@funkwelt.net`** mit:
  - Was passiert ist (Schritt-für-Schritt)
  - macOS-Version (Apfel-Menü → Über diesen Mac)
  - HAM-Tools-Version (steht im DMG-Dateinamen oder unter Apfel-Menü → Über HAM-Tools)

## Wir freuen uns über Feedback

Auch *"Das könnte man so machen…"*-Ideen sind willkommen. Kategorie **"Sonstiges / Mehrere Module"** im Dropdown wählen.
